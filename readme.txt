Woocommerce Phone Number Required Module is very much usefull to make phone number required field in Checkout Billing Methods.
== Description ==

**Woocommerce Phone Number Required Module** *is used to provide help to customer or users who wants to make Phone Number as required field so they can contact their customers with phone number.

Woocommerce Phone Number Required Features:

* Woocommerce Phone Number Required module will work with Woo Commerce for E-commerce Sites.

* Woocommerce Phone Number Required is used to make phone number required field so that they can contact customers with the number added.

* Woocommerce Phone Number Required can be helpfull to send order confirmation on this phone number.

== Installation ==
* Upload FAQ Plugin Folder to the `/wp-content/plugins/` directory 
* Activate the plugin through the 'Plugins' menu in WordPress 
* Check Billing and Shipping address fields with phone number which will become complulsory
